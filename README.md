# Telegram Bot

A simple Telegram bot that responds to /start command.

## Setup

1. Install dependencies:
```bash
pip install -r requirements.txt
```

2. Get your bot token from [@BotFather](https://t.me/botfather) on Telegram

3. Set your token as environment variable:
```bash
export TELEGRAM_BOT_TOKEN=your_token_here
```

4. Run the bot:
```bash
python bot.py
```

## Usage

Send `/start` to your bot and it will reply with "Hello! What can I help you with?"
